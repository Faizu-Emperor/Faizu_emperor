->git stutus
    =>to check the status
->git init
    =>init anew repository in local directory

->git add fliename
    =>add file to staging area
->git add.
    =>add all untracked files and changes to straging area

->git commit -m "new files added"
 => commit this file to respository

 ->git config --global user.email "youremilid@gmail.com"
 ->git config --global user.name "your name"

->git log 
    =>what we change it will show in that.
->git diff
    =>KNow exactly what you have changed 
-> Q is for quiting from the terminal

-> git li-files 
    => list all files available in staging area

->git rm filename 
    =>remove this file from staging area and local directory.

-> git rm --cached fliename
    => you want to keep the file in your hard disk but you don't want git to track this file anymore.

*Branch :-
------------
    ->git Branch
        =>returns all branches

    ->git branch second-branch
        => create another branch whose name is second-branch

    ->git checkout second-branch
        =>switch to second-branch

    ->git checkout -b third-branch
        =>creates and switch to third-branch


-> git remote add origin URL
    =>connect Local repository to remote repository

->git push -u origin master 
    =>push changes to remote repository from local repository 

-> git push --set-upstream origin name

->git clone 
->git pull
